PPopupMenu
---------------------------------------
You can easily create PopupMenu and add items to menu in Processing code.

---------------------------------------
Working Environment
---------------------------------------
Any environment that Processing works
-Mac(OSX)
-Windows
-Linux

---------------------------------------
How to set up new code
---------------------------------------
1.Launch Processing and create a new code.
2.Select "Sketch->Add file...", and add "PPopupMenu.pde".
3.Then, "PPopupMenu.pde" will be added to this directory, automatically.
4.Now, you can use "PPopupMenu" in this code.
